# 运行后端服务

```
python -m venv .venv

.\.venv\Scripts\activate

pip install -r requirements.txt

pip install jupyterlab

pip install pdfplumber

pip install openpyxl

pip install Flask

pip install flask-cors

pip install pandas

pip install mysql-connector-python

python .\backend.py

flask --app backend run
```

# 运行前端服务

```
npm install

npm run serve
```
